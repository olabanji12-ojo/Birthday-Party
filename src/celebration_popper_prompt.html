Implement it as a reusable utility function called firePoppers(origin) that can be called from anywhere in the app. LIBRARY: Use tsParticles confetti cannon via @tsparticles/confetti or the confetti() helper from the tsparticles bundle. Do NOT use canvas-confetti — use tsParticles to stay consistent with the rest of the animation stack. THE firePoppers FUNCTION: Create a reusable function that accepts an origin parameter: "hero" or "rsvp". Export it so both HeroSection and RSVPSection can import and call it. TRIGGER 1 — Hero (@70 stamp moment): Call firePoppers("hero") inside the Framer Motion onAnimationComplete callback of the @70 element — fires the instant the stamp animation finishes.

Navigation
Navbar styleOption B + C combined
Sticky top barVery slim, semi-transparent navy, gold border bottom. Left: celebrant initials "SOO". Right: one gold "RSVP" button. Appears after hero scroll.
Dot navigationSmall gold dots on right edge of screen. One dot per section. Active dot glows/fills gold. Clicking jumps to section smoothly.
Animation System
Page loadGold confetti + sparkle explosion bursts from center using tsParticles. Then hero content fades/drops in dramatically.
Scroll triggerAll sections use Framer Motion whileInView with viewport once: true
Entrance styleElements slide up + fade in. Cards use stagger delay (0.1s apart). Nothing pops in too fast.
RSVP submitConfetti burst fires on successful form submission using tsParticles
CountdownLive ticking. Each number flip uses a subtle scale animation.
LibraryFramer Motion for all scroll + entrance animations · tsParticles for confetti/bokeh effects
